Web Design Curriculum

 

はじめに

Web Design Curriculum へようこそ！

このカリキュラムは、Webデザインに必要な基本の考え方や、制作実践までが学べるスタータープログラムです！

 

このカリキュラムで想定している対象者について

  • HTML, CSS の基本的な知識や記述を理解している
  • Webデザインの初学者である

 

このカリキュラムで身につくスキル

  • Webデザインそのものに関する基本知識（そもそもデザインとは何なのか？デザインの必要性とは？など）
  • Webデザインを行うためのツールの使い方（Illustrator、Photoshop、デザイン共有ツールなどの基本操作）
  • Webデザイン実践スキル（どのようにコンテンツを配置するか？など）
  • 現場でのデザイン実践スキル（クライアントからのヒアリングの仕方、デザイン提案の仕方など）

ツールの使い方に関しては、Adobe製品はAdobe公式のチュートリアル・スターターサポートが充実しているため、それに取り組んでいただくことがメインです。
ケースバイケースとなってしまう細かなツール活用テクニックよりは、基礎であり汎用性のあるデザインに関する様々な考え方を学んでいただくことや、制作実践を重ねていただくことに重きを置いているカリキュラムです。

 

カリキュラムの進め方

このカリキュラムではWebデザインの基本的なトピックについて解説し、いくつかの章では最後にチャレンジ課題を提供します。

チャレンジ課題に取り組んだら、メンターへ提出してください。提出物を確認の上、メンターによるフィードバックコメントを返却します。フィードバックには時間がかかる場合がありますので、コメントを待つ間に次のトピックに着手することを推奨します。

提出方法など詳細はそれぞれの課題に記載があるので、よく読んで提出物を見直してから提出してください。

カリキュラムを進めていく中で自力での解決が困難な問題に直面したり、内容に対して疑問点がある時はLINEや、各デザインツールにコメントを記載してください！

 

それでは、はりきって進めていきましょう！！
